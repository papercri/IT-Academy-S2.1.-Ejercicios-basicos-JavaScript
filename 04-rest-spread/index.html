Ejercicio 2

Operador Rest en Funciones: Crea una función 'suma' que utilice el operador rest para aceptar un número indeterminado de argumentos y devolver su suma.

                function suma(...arr) {
                    let total = 0;
                    for (let a of arr) {
                        total += a;
                    }
                    return total;
                    }
                    console.log(suma(1,2,3,12));
